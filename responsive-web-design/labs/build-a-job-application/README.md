# User Stories:

1. You should have a div element with the class container.[x]
2. Inside the div element, you should have a form element.[x]
3. The form should contain an input element with the type text and the id name for entering the user's full name.[x]
4. You should have another input element with the type email and the id email for entering the user's email address.[x]
5. The form should include a select element with the id position that allows users to select a job position.[x]
6. You should have a fieldset element with class of radio-group.[x]
7. Inside .radio-group you should have a set of input elements with the type radio and relevant labels for selecting availability options (e.g., Full-Time, Part-Time). The group name should be availability. [x]
8. You should have a textarea element with the id message for entering a message.[x]
9. You should associate every input element with a label element.[x]
10. You should have a button element with the type submit for submitting the form.[x]
11. Add a :focus pseudo-class to the input and textarea elements to change their border color when focused.[x]
12. The input, select and textarea elements should have an :invalid pseudo-class that changes the border color to red when invalid input is detected.[x]
13. The input, select and textarea elements should have a :valid pseudo-class that changes the border color to green when valid input is entered.[x]
14. The button element should have a :hover pseudo-class that changes the background color when hovered over.[x]
15. Use the :checked pseudo-class on .radio-group input[type="radio"] to add a border color, background color and a box shadow when the radio button is selected. [x]
16. Use the :checked pseudo-class on radio buttons to change the text color of the associated label when the option is selected.
17. Add a :first-of-type pseudo-class to the input element to style the first input field differently. (e.g., rounded corners).[x]
